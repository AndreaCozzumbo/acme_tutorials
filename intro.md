# ACME Tutorials

Welcome to the ACME Tutorials site! This collection provides hands-on tutorials for multi-messenger astronomy and related topics.

## Contents

- **GWFish**: Gravitational wave Fisher matrix analysis
- **Alerts**: Real-time alert streaming and processing
- **RAVEN**: Computing the significance of multi-messenger candidates
- **BAT-GLIMPSE**: Gamma-ray Localization using Imaging and Mosaic techniques for Pointing and Slew Epochs
- **MAGGPY**: connecting neutron star margers and short GRB populations

## Interactive Notebooks

All notebooks in this site can be run interactively using Binder. Click the 🚀 rocket icon at the top of any page and select "Binder" to launch an interactive session.

## Getting Started

Browse the tutorials using the navigation menu on the left, or click on any of the topics above to jump directly to a specific tutorial.
